# Tailwind CSS v2.0 Crash Course

Tailwind CSS v2.0 Crash Course

The code in this repository is based on the
[Tailwind CSS 2.0 Crash Course | First impressions]()
video.

## Setup

```bash
npm init -y
npm install tailwindcss postcss autoprefixer
npm install postcss-cli
# create postcss.config.js before next steps
npx tailwindcss init
# create dev script in pagckage.json before next steps
npm run dev
npm install -D postcss-cli
```
